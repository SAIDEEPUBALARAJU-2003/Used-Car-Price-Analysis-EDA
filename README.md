## Project Overview
This project focuses on analyzing used car prices to understand how different factors influence car pricing in the resale market.

## Objective
- Identify key features affecting used car prices
- Help buyers make informed decisions
- Help sellers price cars competitively

## Data Source
- Data scraped from Cars24 website

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Regular Expressions
- Web Scraping (Requests, BeautifulSoup)

## Key Features Analyzed
- Brand
- Model
- Manufacturing Year
- Kilometers Driven
- Fuel Type
- Owner Type
- Price

## Key Insights
- Car price decreases as car age increases
- Diesel cars are generally priced higher than petrol cars
- Premium brands retain higher resale value
- Most cars fall in the price range of 2.5 to 6.5 lakhs
